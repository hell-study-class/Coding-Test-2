## 📚 Study Guideline
1. 각자의 브랜치를 만들어 문제 풀이를 진행한다
2. 대면 활동 전 PR을 생성하고, 각자가 푼 문제를 정리해온다

## 🙆🏻‍♂️ Git Commit, PR Message Convention
### Commit Type
```bash
[태그] {난이도}:{문제번호} {알고리즘 유형}
```
- 예시: `[baekjoon] S3:1051 implementation`

### File
- {난이도}:{문제번호}
- 예시: `S3:1051`


#### 🚉 플랫폼 약어
| 플랫폼 | 태그 |
| :--: | :--: |
| 백준 | [baekjoon] |
| 프로그래머스 | [programmers] |
| 리트코드 | [leetcode] |
| 코드트리 | [codetree] |

### Pull Request
```bash
[날짜] 이름_n회차 문제 풀이
```
- 예시: `[20240627] 구태형_1회차 문제 풀이`
